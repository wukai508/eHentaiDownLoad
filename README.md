## ⚙️ 核心功能说明

### 页面解析
- 自动识别 E-Hentai 漫画目录页结构
- 提取漫画标题、页数、Gallery ID
- 解析所有缩略图链接（支持多页）

### 图片获取
- 访问每个图片展示页
- 提取真实图片地址
- 支持多种页面结构识别

### 下载控制
- Service Worker 后台下载
- 500ms 间隔防止触发反爬虫
- 实时进度更新到 UI
- 支持取消下载

### 文件命名
- 自动清理文件名中的非法字符
- 图片按序号命名：`001.jpg`, `002.webp` 等
- ZIP 文件与文件夹同名

## ⚠️ 注意事项

1. **合法使用**：本插件仅供学习研究使用，非官方提供，请遵守相关法律法规
2. **下载间隔**：默认 500ms 间隔，请勿修改过小以免被封禁
3. **合理使用**：此插件的目的是用于下载一些没有种子或者只有几页的本子，尽量还是使用种子或者官方存档下载
4. **文件大小**：打包下载会将所有图片加载到内存，大型画册可能占用较多内存
5. **浏览器兼容**：仅支持基于 Chromium 的浏览器（Chrome、Edge、Brave 等）


## 🚀 使用方法

### 第一步：打开漫画页面
在浏览器中访问 E-Hentai 漫画目录页

### 第二步：启动插件
点击浏览器工具栏中的插件图标，会在新标签页打开下载界面

### 第三步：确认信息
- 查看漫画名称、图片数量等信息
- 可点击 ✏️ 按钮编辑保存的文件夹名称

### 第四步：选择下载方式
- **直接下载**：图片保存到 `下载/漫画名称/` 文件夹
- **打包下载**：所有图片打包为 `漫画名称.zip` 文件

### 第五步：开始下载
点击「🚀 开始下载」按钮，等待下载完成
